## Установка
1.  Клонируйте репозиторий:
    `git clone <url репозитория>`
2.  Перейдите в каталог:
    `cd название-проекта`
3.  Установите зависимости:
    `npm install` [2]

## Использование
Запустите приложение с помощью:
`npm start`
